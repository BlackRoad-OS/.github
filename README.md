## What Is BlackRoad OS?

BlackRoad is a **routing company, not an AI company.** We connect users to intelligence that already exists — Claude, GPT, Llama, NumPy, legal databases — through an orchestration layer we own.

> **The insight:** Intelligence is already trained. The value is in routing requests to the right tool at the right time.
