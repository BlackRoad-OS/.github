## 🤔 What is BlackRoad?

**BlackRoad routes you to AI — it doesn't try to be AI.**

Think of it like a smart switchboard. You ask a question or run a task, and BlackRoad figures out whether to send it to Claude, GPT, a local model, a spreadsheet, or a Raspberry Pi in the next room. You don't need to know which one — it just works.

- 🔀 **Routes requests** to the right AI/tool automatically
- 🖥️ **Runs on a Pi cluster** you can hold in your hand (~$40/month total)
- 🎮 **Has a pixel OS** you can actually play with right now (☝️ the links above)
- 🤖 **12 named AI agents** — each specialized for different tasks
