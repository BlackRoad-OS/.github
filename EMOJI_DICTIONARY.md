## Quick Reference

### Traffic Light System

```
:green_circle: = GO (production ready)
:yellow_circle: = CAUTION (needs review)
:red_circle: = STOP (do not use)
:blue_circle: = ARCHIVED (historical)
```

### Priority Levels

```
:fire: P0 = Critical (drop everything)
:warning: P1 = High (today)
:yellow_circle: P2 = Medium (this week)
:blue_circle: P3 = Low (when possible)
```

### Deployment Status

```
:rocket: = Deploying
:white_check_mark: = Deployed
:x: = Failed
:recycle: = Rolling back
```
